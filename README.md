# Global Earthquakes

A react app to visualize an earthquake dashboard based on the time preferences

## Run the project

clone the project

select master branch

Navigate to the root folder:

```
npm install
```

```
npm start
```

browser listens to [http://localhost:3000](http://localhost:3000) as default port
